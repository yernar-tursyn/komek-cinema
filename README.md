# Komek Cinema

Веб-приложение для кинотеатра на Laravel.

## Описание

Система для отображения афиши фильмов, расписания сеансов и информации о кинотеатре.

## Технологии

- **Laravel 10** — PHP фреймворк
- **PHP 8.1+**
- **Blade** — шаблонизатор
- **Vite** — сборщик фронтенда

## Установка

1. Установите зависимости:
```bash
composer install
npm install
```

2. Скопируйте файл окружения:
```bash
cp .env.example .env
```

3. Сгенерируйте ключ приложения:
```bash
php artisan key:generate
```

4. Настройте базу данных в `.env`

5. Запустите миграции:
```bash
php artisan migrate
```

6. Запустите сервер разработки:
```bash
php artisan serve
npm run dev
```
